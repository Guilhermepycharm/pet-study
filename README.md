# Pet Estudos

Aplicativo de cronograma de estudos para o ENEM com revisoes espacadas, metas, foco semanal e acompanhamento por pet virtual.

## Rodar localmente

**Pre-requisito:** Node.js

1. Instale as dependencias:
   `npm install`
2. Inicie o servidor de desenvolvimento:
   `npm run dev`

O app abre por padrao em `http://localhost:3000/`.
